Skip List Visualisation

Reasoning about the internals of a skip list can be quite hard. This skip list implementation has some methods that can write to a stream in the Graphviz DOT format. This data can be used to create PNG or SVG files.

The images here are meant to be read from bottom to top.

Examples

Insert, Remove

Insert, Remove and Repeat

This diagram shows the insertion of the values 0 to 4 in order followed by removal in reverse order. This is repeated four times, note how different the shape is each time even though the data is the same. Diagram

Coin Toss Probability of 25%

The normal skip list uses a fair coin. This shows what would happen if the coin had only a 25% chance of adding a layer to the alternate lists when inserting of the values 0 to 7. Diagram

Coin Toss Probability of 75%

The normal skip list uses a fair coin. This shows what would happen if the coin had a 75% chance of adding a layer to the alternate lists when inserting of the values 0 to 7. As expected the number of levels is much higher. Diagram
